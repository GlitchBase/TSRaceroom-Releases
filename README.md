# TS Raceroom

An unofficial companion & race-prep app for **Touge Shakai** — leaderboards, matchup planning, PB tracking, a track-map digitizer, and an in-game battle HUD mod.

> This repository hosts the **public releases** of TS Raceroom. The source code lives in a private repository; only built installers and release notes are published here.

## Download

**Latest version: `v0.9.43`**

Grab the installer from the [**latest release**](https://github.com/GlitchBase/TSRaceroom-Releases/releases/latest):

- `TS-Raceroom-Setup-0.9.43.exe` — Windows installer

### Bridge mod (MelonLoader)

The in-game HUD mod can update between app versions. Prefer **Game Link → Install / Update / Repair** in the desktop app.

Manual drop-in: from the [`mod-latest`](https://github.com/GlitchBase/TSRaceroom-Releases/releases/tag/mod-latest) release, copy `TSRaceroomBridge.dll` into `<Touge Shakai>/Mods/`.

## Install

1. Download `TS-Raceroom-Setup-0.9.43.exe` from the latest release.
2. Run the installer and follow the prompts (you can choose the install directory).
3. Launch **TS Raceroom** from the Start menu or desktop shortcut.

Windows SmartScreen may warn about an unrecognized publisher because the app is not code-signed. Choose **More info → Run anyway** to continue.

## Auto-updates

The app checks this repository for new releases and updates itself automatically. When a new version is published here, you'll be prompted to install it on next launch.

## Features

- **Leaderboards** — browse and cache Touge Shakai time-attack boards per track, layout, and weather.
- **Matchup planning** — build team matchups, scout opponents, and plan round lineups.
- **PB Wall** — track personal-best lap-time changes for the drivers you follow.
- **Track-map digitizer** — trace track maps from images and export them for use in the app.
- **In-game battle HUD** — a MelonLoader mod that shows the time to beat, your team's best, and your last lap with live sector splits, directly in-game.
- **Cloud sync** — optional Steam sign-in to sync your data across devices.

## System requirements

- Windows 10 or 11 (64-bit)
